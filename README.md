# Battleship

A browser-based Battleship game built with TypeScript and Vite.

## Live Demo

Play it here: **[https://dist-ewhjwoql.devinapps.com](https://dist-ewhjwoql.devinapps.com)**

## Development

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Testing

```bash
npm test
```

## Project Structure

- `src/engine/` — core game logic
- `src/ai/` — computer opponent
- `src/ui/` — DOM rendering and input
- `src/sound/` — audio effects
- `tests/` — Vitest test suites
